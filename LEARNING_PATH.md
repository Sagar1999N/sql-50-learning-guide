# Learning Path

## Recommended Study Order:
1. **Basic SQL** - Foundation first
2. **Advanced Joins** - Multiple table relationships
3. **Window Functions** - Analytical queries
4. **Subqueries & CTEs** - Alternative approaches
5. **Advanced Patterns** - Complex problem solving

## Time Allocation:
- Basic SQL: 1 week
- Advanced Joins: 1 week
- Window Functions: 2 weeks
- Subqueries & CTEs: 1 week
- Advanced Patterns: 1 week
